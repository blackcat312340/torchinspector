    Constructor signature: __init__(self, model: nn.Module, optimizer: torch.optim.Optimizer, backend: TensorBoardBackend, monitor: TrendMonitor, log_interval: int = 100, micro_batch_variance: bool = False, analysis_interval: int = 5000)

    Instance state:
    - _model, _optimizer, _backend, _monitor, _log_interval
    - _micro_batch_variance: bool
    - _analysis_interval: int

    collect(self, step: int, batch_inputs: torch.Tensor | None = None, batch_targets: torch.Tensor | None = None, loss_fn: object | None = None) method:
    1. If step % _log_interval != 0: return (interval gating)
    2. Compute total gradient norm independently per D-03:
       - total_norm_sq = 0.0
       - For name, param in self._model.named_parameters():
         - If param.grad is None: continue
         - grad = param.grad.detach().float()
         - If grad.isnan().any() or grad.isinf().any(): continue
         - total_norm_sq += grad.norm(p=2).item() ** 2
       - grad_norm = total_norm_sq ** 0.5
    3. Append grad_norm to _grad_norm_window; if len > 100, pop(0) (D-04)
    4. Compute GNS if len(_grad_norm_window) >= 10:
       - lr = self._optimizer.param_groups[0]["lr"]
       - batch_size = max(1, sum(1 for _, p in self._model.named_parameters() if p.grad is not None))
       - gns = np.var(self._grad_norm_window) * lr / batch_size (D-01)
       - Write batch_sensitivity/gns = gns to backend
       - Call self._monitor.check_bsz(gns, step)
    5. Micro-batch variance analysis: if self._micro_batch_variance AND step % self._analysis_interval == 0 AND batch_inputs is not None AND batch_targets is not None AND loss_fn is not None:
       - batch_size = batch_inputs.shape[0]
       - If batch_size < 4: skip (Pitfall 1)
       - saved_training = self._model.training (D-14)
       - try:
         - self._model.eval() (D-13)
         - input_chunks = torch.chunk(batch_inputs, 4, dim=0)
         - target_chunks = torch.chunk(batch_targets, 4, dim=0)
         - micro_grad_norms = []
         - For each chunk pair (inp, tgt):
           - self._optimizer.zero_grad()
           - output = self._model(inp)
           - loss = loss_fn(output, tgt)
           - loss.backward()
           - chunk_norm_sq = sum(p.grad.detach().float().norm(p=2).item() ** 2 for p in self._model.parameters() if p.grad is not None)
           - micro_grad_norms.append(chunk_norm_sq ** 0.5)
         - micro_var = float(np.var(micro_grad_norms))
         - Write batch_sensitivity/micro_batch_variance = micro_var
       - finally:
         - self._model.train(saved_training) (restore regardless of exception)
       - After micro-batch analysis: re-run forward+backward on full batch to restore gradients (the micro-batch passes clobber them)

    close(self) method: no-op (pass), consistent with pattern.

    IMPORTANT: After micro-batch variance analysis, gradients from the original batch are destroyed. The method should document this side effect. Users should call optimizer.zero_grad() + loss.backward() after if they need gradients, or the Inspector wiring (Plan 02) will handle this by calling micro-batch analysis AFTER the main gradient collection.
