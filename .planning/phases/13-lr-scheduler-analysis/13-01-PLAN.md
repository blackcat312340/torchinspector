    Create src/torchinspector/collectors/lr_scheduler.py containing class LRCollector.

    Constructor signature: __init__(self, optimizer: torch.optim.Optimizer, backend: TensorBoardBackend, monitor: TrendMonitor, log_interval: int = 100, warmup_steps: int = 100)

    Instance state:
    - _optimizer, _backend, _monitor, _log_interval, _warmup_steps
    - _prev_lr: float | None = None
    - _anomaly_window_active: bool = False
    - _anomaly_window_steps: int = 0
    - _anomaly_window_losses: list[float] = []
    - _anomaly_start_step: int = 0

    collect(self, step: int, loss_val: float | None = None) method:
    1. Read current_lr = self._optimizer.param_groups[0]["lr"] (D-13: only group 0)
    2. If _prev_lr is None: set _prev_lr = current_lr, write lr/anomaly=0.0, return
    3. If step < _warmup_steps: set _prev_lr = current_lr, write lr/anomaly=0.0, return (D-04)
    4. Compute ratio = current_lr / _prev_lr (guard _prev_lr <= 0 -> skip)
    5. Detect anomaly: ratio > 10.0 -> "lr_spike", ratio < 0.01 -> "lr_drop", else None (D-01, D-03)
    6. If anomaly detected:
       a. Write lr/anomaly = 1.0 (spike) or -1.0 (drop) (D-02: single WARN)
       b. Call self._monitor.check_lr(current_lr, step)
       c. Reset and start loss window: _anomaly_window_active=True, _anomaly_window_steps=0, _anomaly_window_losses=[]
       d. If loss_val is not None and math.isfinite(loss_val): append to _anomaly_window_losses
    7. Else (no anomaly):
       a. Write lr/anomaly=0.0
       b. If _anomaly_window_active and loss_val is not None and math.isfinite(loss_val):
          - Append loss_val to _anomaly_window_losses
          - Increment _anomaly_window_steps
          - If _anomaly_window_steps >= 50 (D-06): call _finalize_loss_response(step)
    8. Set _prev_lr = current_lr

    _finalize_loss_response(self, step: int) method:
    1. If len(_anomaly_window_losses) < 2: reset window, return
    2. initial = _anomaly_window_losses[0], final = _anomaly_window_losses[-1]
    3. If initial == 0: reset window, return
    4. pct_change = ((final - initial) / abs(initial)) * 100
    5. Write lr_response/loss_change_pct = pct_change (D-07)
    6. If pct_change >= 0: call self._monitor.check_lr_stagnation(step) (D-08)
    7. Reset window

    _reset_anomaly_window(self) method: sets _anomaly_window_active=False, _anomaly_window_steps=0, _anomaly_window_losses=[]

    close(self) method: no-op (pass), consistent with pattern

    - TestCollectWrites: test writes lr/anomaly=0.0 normally, test writes lr/anomaly=1.0 on spike, test writes lr/anomaly=-1.0 on drop
    - TestClose: test close is no-op, test close idempotent
