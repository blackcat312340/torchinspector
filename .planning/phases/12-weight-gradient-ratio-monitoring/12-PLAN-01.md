# Phase 12 Plan 01: WeightGradRatioCollector Core

**Requirements:** WGR-01 (per-layer W/G ratio), WGR-03 (log-space ratio)
**Goal:** Implement the core collector that computes per-layer log-space weight/gradient ratios with backward hook gradient caching.

---

## Context

The existing `GradientCollector` (`src/torchinspector/collectors/gradient.py`) computes per-parameter gradient norms and update ratios (`||grad|| / (||weight|| + eps)`). It reads `param.grad` directly, which works when `inspector.step()` is called before `optimizer.zero_grad()` (Order A). However, many training loops zero gradients before `inspector.step()` (Order B), leaving `param.grad` as None.

The new `WeightGradRatioCollector` solves this by registering `register_full_backward_hook()` on watched modules to cache gradient norms *during* the backward pass, before `zero_grad()` can clear them. It then computes the log-space ratio `log(||w||+eps) - log(||grad||+eps)` at collection time using the cached gradient norms.

---

## Task 1: Create `WeightGradRatioCollector` class skeleton

**File:** `src/torchinspector/collectors/weight_grad_ratio.py` (NEW)

Create the module with:

```python
"""Weight/Gradient ratio collector ŌĆö per-module log-space W/G ratios."""

from __future__ import annotations

import math
from typing import TYPE_CHECKING

from torch import nn
from torch.utils.hooks import RemovableHandle

if TYPE_CHECKING:
    from torchinspector.backends.tensorboard import TensorBoardBackend
    from torchinspector.hooks import HookManager

_EPS = 1e-8
```

Implement `__init__` with parameters:
- `model: nn.Module`
- `hook_manager: HookManager`
- `backend: TensorBoardBackend`
- `log_interval: int = 100`

Initialize:
- `self._model`
- `self._hook_manager`
- `self._backend`
- `self._log_interval`
- `self._grad_norm_cache: dict[str, float] = {}`
- `self._backward_handles: list[RemovableHandle] = []`

## Task 2: Implement backward hook registration

**File:** `src/torchinspector/collectors/weight_grad_ratio.py`

### `_make_backward_hook(name: str) -> Callable`

Create a closure that:
1. Iterates `module.named_parameters(recurse=False)` ŌĆö only this module's direct parameters
2. For each param with `param.grad is not None`:
   - Detach and cast to float32: `g = param.grad.detach().float()`
   - Check finiteness: `if g.isfinite().all()`
   - Accumulate squared norm: `total_norm_sq += g.norm(p=2).item() ** 2`
3. If `total_norm_sq > 0`, cache the L2 norm: `self._grad_norm_cache[name] = total_norm_sq ** 0.5`

### `_ensure_hooks(watched: set[str]) -> None`

1. Get `modules = dict(self._model.named_modules())`
2. For each `name` in `watched`:
   - Skip if already handled (check against a set of names from existing handles)
   - Get `module = modules[name]`
   - Register: `handle = module.register_full_backward_hook(self._make_backward_hook(name))`
   - Append handle to `self._backward_handles`

**Design decision:** Backward hooks are registered in the collector itself (not HookManager). This keeps hook management self-contained per collector. The HookManager's forward hooks serve a different purpose (activation caching).

## Task 3: Implement log-space ratio computation

**File:** `src/torchinspector/collectors/weight_grad_ratio.py`

### `_compute_log_ratio(weight_norm: float, grad_norm: float) -> float`

```python
return math.log(weight_norm + _EPS) - math.log(grad_norm + _EPS)
```

Properties:
- Positive: weight dominates (vanishing risk)
- Negative: gradient dominates (exploding risk)
- Zero: balanced
- No overflow/underflow due to log compression and eps protection

## Task 4: Implement per-module collection with mean + max aggregation

**File:** `src/torchinspector/collectors/weight_grad_ratio.py`

### `_collect_for_module(name: str, module: nn.Module, step: int) -> None`

1. Collect ratios for each parameter in the module:
   ```python
   ratios = []
   for param_name, param in module.named_parameters(recurse=False):
       if param.grad is None:
           continue
       w_norm = param.detach().float().norm(p=2).item()
       g_norm = self._grad_norm_cache.get(name)  # module-level cached grad norm
       if g_norm is None:
           continue
       if w_norm < _EPS and g_norm < _EPS:
           continue  # Both negligible ŌĆö skip
       ratio = self._compute_log_ratio(w_norm, g_norm)
       ratios.append(ratio)
   ```
2. If no ratios collected, return early
3. Compute aggregates:
   - `mean_ratio = sum(ratios) / len(ratios)`
   - `max_ratio = max(ratios)`
4. Write to backend:
   - `self._backend.write_scalar(f"ratios/{name}/mean", mean_ratio, step)`
   - `self._backend.write_scalar(f"ratios/{name}/max", max_ratio, step)`
5. Feed to TrendMonitor (for Plan 02 integration):
   - `self._monitor.check_wgr(name, mean_ratio, step)` ŌĆö but since Plan 02 adds this method, just feed via `self._monitor.check(...)` with the existing API for now. In Plan 02 we'll upgrade to `check_wgr()`.

## Task 5: Implement `collect()` and `close()` lifecycle

**File:** `src/torchinspector/collectors/weight_grad_ratio.py`

### `collect(step: int) -> None`

1. Guard: `if step % self._log_interval != 0: return`
2. Get watched set: `watched = set(self._hook_manager._handles.keys())`
3. If empty, return
4. Call `self._ensure_hooks(watched)`
5. Iterate `self._model.named_modules()`:
   - Skip root module (name == "")
   - Skip if name not in watched
   - Call `self._collect_for_module(name, module, step)`

### `close() -> None`

1. Remove all backward handles: `for h in self._backward_handles: h.remove()`
2. Clear: `self._backward_handles.clear()` and `self._grad_norm_cache.clear()`
